### geo-leaflet
Spring Boot приложение с функционалом выделения полигона на карте и выведения его координат в консоль.

### Логика работы
Бэкенд приложения (Spring) обрабатывает пользовательские запросы и служит входной точкой к карте мира.

После чего карта инициализируется координатами Москвы с помощью JavaScript.

При выделении полигона на карте срабатывает событие L.Draw.Event.CREATED, в обработчике которого JS получает координаты полигона и отправляет их с помощью AJAX-запроса по эндпойнту /polygon.

В самом конце бэкенд принимает POST-запрос по данному эндпойнту и выводит координаты полигона в консоль.
